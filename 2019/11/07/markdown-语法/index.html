语法

图片title为当鼠标移到图片上方时，显示的文字，title可加可不加

7列表

无序列表

可以用”*”、”+”、”-“(需用空格与后面的内容隔开)

例

* 无序列表1
+ 无序列表2
- 无序列表3

效果

  • 无序列表1
  • 无序列表2
  • 无序列表3

有序列表

数字加上”.”

例

1.有序列表1
2.有序列表2
3.有序列表3

效果

1.有序列表1
2.有序列表2
3.有序列表3
